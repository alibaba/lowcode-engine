## 模块简介

通过该模块可以开发应用级低代码设计器。

## 变量

### isActive

是否启用 workspace 模式

### window

当前设计器窗口模型

关联模型 [IPublicModelWindow](./model/window)

## 方法签名

注册资源

/** 注册资源 */
