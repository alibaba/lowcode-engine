#### 简介
用于输入数字。

#### setter 配置
| 属性名 | 说明 |
| --- | --- |
| min, max | 指定最大最小值 |
| defaultValue | 默认值 |
| step | 指定步长 number |
| units | 指定单位 string |
| precision | 设置小数位数 number |

#### 返回类型
#### Number
会返回value值
