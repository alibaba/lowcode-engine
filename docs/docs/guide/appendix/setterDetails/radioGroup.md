#### setter 配置
| 属性名 | 说明 |
| --- | --- |
| defaultValue | 默认值 |
| options | 传入的数据源，
**参数格式**:
[
{img: 'url',
value: 'text',
label/title: 'text'}, ...
]

[ 'text', 'text', ...] |
