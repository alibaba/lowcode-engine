## 简介

用于直观的展示选择并选择。

## setter 配置

| 属性名       | 说明                                                                                 |
| ------------ | ------------------------------------------------------------------------------------ | ----------------------- |
| defaultValue | 默认值                                                                               |
| options      | 传入的数据源，**参数格式**: `[{img: 'url', value: 'text', label/title: 'text'}, ...] | [ 'text', 'text', ...]` |

## 返回类型

String | Number | Boolean
会返回 options 中的 value 值
