## 安装运行

```bash
# 安装依赖
tnpm install

# 启动调试
npm start

# 本地构建（一般来说不需要）
npm run build

# 日常部署：将资源发布到日常 CDN
npm run deploy

# 线上部署：将资源发布到线上 CDN
npm run deploy:online
```
